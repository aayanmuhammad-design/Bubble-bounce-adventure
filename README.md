# Tetraris
**Tetraris** is a fast-paced Tetris-style puzzle game built using Python and the Pygame library, where players must strategically arrange falling blocks to complete horizontal lines and score points. The game features smooth controls, a scoring and level system that increases difficulty over time, and mechanics like hard drop and a ghost piece to help players plan their moves more effectively. To play, use the left and right arrow keys to move pieces, the up arrow to rotate them, and the down arrow to speed up their fall. Press the spacebar to instantly drop a piece into place for extra points. The objective is to clear as many lines as possible without letting the stack reach the top of the grid, which ends the game
